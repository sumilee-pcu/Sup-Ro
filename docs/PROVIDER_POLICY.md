# 지도·공공데이터 공급자 정책 메모

확인 기준일은 2026-07-14이다. 이 문서는 법률 자문이나 공급자 계약을 대체하지 않으며 공개 배포와 유료 기능 활성화 직전에 각 공식 문서에서 다시 확인한다.

## Kakao Maps·Local 활성화 기록

| 항목                | 현재 설정                                                                                     |
| ------------------- | --------------------------------------------------------------------------------------------- |
| 개발자 앱           | `Sup-Ro`, 소유자 권한, 교육 카테고리                                                          |
| 지도 제품           | 2026-07-14 `ON`                                                                               |
| 계정 내 활성화 순서 | 대시보드에서 확인된 3개 기존 앱은 모두 지도 `OFF`; `Sup-Ro`가 첫 번째이자 유일한 지도 활성 앱 |
| JavaScript 도메인   | `http://localhost:3000`, `https://sup-ro.vercel.app` 등록                                     |
| JavaScript 키       | `Sup-Ro Local Web`; 브라우저용, ignored `.env.local`에만 값 보관                              |
| REST API 키         | `Sup-Ro Server Local`; 서버 전용, ignored `.env.local`에만 값 보관                            |
| IP 제한             | REST 키를 공개 배포에 사용하지 않아 미설정; 라이브 서버 연동 시 고정 송신 IP를 우선 검토      |
| 결제 설정           | 유료 API·BizWallet 미활성화                                                                   |

실제 키 값, 다른 앱의 식별자, 계정 정보는 Git 추적 문서에 기록하지 않는다. JavaScript 키에는 확정된 HTTPS 운영 도메인만 추가하고 `localhost`는 로컬 개발을 위해 유지한다.

## 현재 무료 쿼터와 초과 단가

공식 [쿼터 문서](https://developers.kakao.com/docs/ko/getting-started/quota)의 2026-07-14 표시값을 기준으로 한다. 쿼터는 앱 단위로 합산되며, 유료 API를 별도로 활성화하지 않은 현재 설정에서는 무료 한도 초과 시 과금 대신 실패를 감지해 fixture로 전환한다.

| API                       |       무료 쿼터 | 유료 API 활성화 시 공지 단가 |
| ------------------------- | --------------: | ---------------------------: |
| 전체 API 월 합계          |  3,000,000회/월 |              API별 단가 적용 |
| Kakao Map JavaScript      |    300,000회/일 |                     0.1원/회 |
| Local 키워드 검색         |    100,000회/일 |                       2원/회 |
| Local 카테고리 검색       |    100,000회/일 |                       2원/회 |
| Local 주소·좌표 변환 계열 | 각 100,000회/일 |                     0.5원/회 |

Kakao가 공지한 [2026-07-21 신규 지도 API·무료 쿼터 정책](https://devtalk.kakao.com/t/api-notice-on-new-kakao-map-api-features-and-free-quota-policy/150222)에 따르면 계정에서 처음 지도 기능을 활성화한 앱에는 무료 쿼터가 제공되고, 두 번째 이후 앱은 BizWallet 연결이 요구될 수 있다. `Sup-Ro`는 시행 전 활성화했고 현재 계정의 첫 지도 활성 앱임을 확인했다. 신규 대중교통·도보·자전거 경로 API의 공지 무료 한도는 각각 1,000회/일, 유료 단가는 10원/회다. 실제 어댑터 구현은 2026-07-21 공개될 최종 명세를 다시 확인한 뒤 진행하며 유료 기능은 자동 활성화하지 않는다.

## 표시·캐시·보안 경계

- 지도 JavaScript 키만 `NEXT_PUBLIC_KAKAO_MAP_APP_KEY`로 브라우저에 전달한다.
- Local·Mobility REST 키는 서버 전용이며 응답·로그·문서에 포함하지 않는다.
- 지도, 장소, 경로 공급자 ID를 내부 장소 ID와 분리한다.
- [Kakao Maps API 공통 문서](https://developers.kakao.com/docs/ko/kakaomap/common)에 따라 JavaScript SDK에는 REST 키가 아닌 JavaScript 키와 등록 도메인을 사용한다.
- 지도 SDK가 생성하는 Kakao 로고·저작권 표시는 숨기거나 변형하지 않는 것을 UI 구현 기준으로 둔다.
- 원시 Local 응답을 장기 저장하거나 재배포하지 않는다. 현재 구현은 정책 검토가 끝나지 않은 라이브 원시 응답의 영속 캐시를 기본 사용하지 않으며, 정규화 결과를 저장해야 할 때는 공개 전 법률·약관 검토를 거친다.
- 401·403·429 또는 네트워크 실패를 성공으로 위장하지 않고, 데이터 모드와 실패 이유를 표시한 뒤 호환 fixture로만 전환한다.
- 실제 전세버스 운행 가능성, 승하차 위치, 통행 제한은 지도 경로만으로 확정하지 않는다.

## 기상·대기질·안전시설

- 공공데이터 포털 또는 기관 키는 서버 전용으로 보관한다.
- 측정값과 예보값을 구분하며 조회 시각과 예보 유효 범위를 기록한다.
- 자동 조회는 안전 보장이 아니다. 학교 절차, 기상특보, 장소·운송기관 공식 안내를 함께 확인한다.
- 응급·안전시설 위치는 참고 정보로만 표시하고 현장 대응 기관을 자동 지정하지 않는다.

## 캐시·장애 처리

1. 공급자 정책이 재사용을 허용한 경우에만 캐시를 읽는다.
2. 만료 캐시는 최신 사실처럼 표시하지 않고 `cache`와 만료 경고를 함께 반환한다.
3. 라이브 실패 시 호환 캐시가 없으면 명시적으로 버전 고정된 `fixture`만 사용한다.
4. fixture 전환은 실패 원인, 데이터 모드, 사용 불가능한 정보를 사용자에게 표시한다.
5. 테스트는 무제한 실 API 호출 대신 정책상 허용되는 최소 기록 응답과 합성 fixture를 사용한다.
