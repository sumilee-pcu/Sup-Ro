# 수업로 AI 네이밍 결정

## 결정

| 항목          | 값                         |
| ------------- | -------------------------- |
| 한글 브랜드   | 수업로 AI                  |
| 영문 표기     | SueopRo AI                 |
| 저장소 슬러그 | `sueop-ro`                 |
| 첫 서비스     | AI 체험학습 코디네이터     |
| 태그라인      | 교실 밖 배움을 하나의 길로 |
| 상태          | 공개 전 작업 브랜드        |

`수업로`는 수업과 길을 결합한 이름이다. 사용자가 성취기준에서 출발해 장소, 이동, 안전, 활동지까지 하나의 배움 경로를 만든다는 제품 구조를 직접 표현한다. 체험학습에 한정되지 않아 캠퍼스 접근성, 지역안전, 교육격차, 진로체험 서비스로 확장할 수 있다.
