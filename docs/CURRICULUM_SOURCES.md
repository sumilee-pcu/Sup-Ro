# 교육과정 출처와 지원 범위

## 기준 출처

초기 기준은 국가교육과정정보센터(NCIC)의 [2022 개정 초·중등학교 교육과정 고시 안내](https://ncic.go.kr/board/B0031.cs?act=read&bwrId=1271&pageIndex=1&pageUnit=15)다. 해당 안내는 교육부 고시 제2022-33호와 학교급·교과별 첨부자료를 연결한다. 교육부의 [2022 개정 교육과정 확정·발표](https://www.moe.go.kr/boardCnts/viewRenew.do?boardID=294&boardSeq=93459&lev=0)도 정책 배경 확인에 사용한다.

2026학년도 적용 범위를 고려해 첫 대표 시나리오는 중학교 2학년 과학으로 정했다. 학교급별 실제 적용 시점과 고시 정정 여부는 운영 전에 NCIC에서 다시 확인해야 한다.

## MVP 지원 범위

| 항목            | 현재 범위                       | 상태                  |
| --------------- | ------------------------------- | --------------------- |
| 교육과정        | 2022 개정 교육과정              | 공식 고시 링크 연결   |
| 학교급          | 초등학교·중학교·고등학교 스키마 | 대표 fixture는 중학교 |
| 대표 학년·교과  | 중학교 2학년 과학               | 구현·평가 완료        |
| 성취기준 데이터 | 짧은 수업 설계용 합성 요약      | 원문 인용 아님        |
| 검색            | 학교급·학년·교과·키워드·식별자  | 로컬 버전 데이터셋    |

## 저작권과 사실성 경계

- 저장소의 `DEMO-*` 식별자는 공식 성취기준 코드가 아닌 시연용 식별자다.
- fixture의 `sourceText`는 고시 원문을 복제하지 않은 수업 목표 요약이다.
- `generatedInterpretation`은 수업로 AI가 사용하는 수업 설계 해석이며 공식 원문과 분리한다.
- 공식 연계로 표시하려면 출처 URL, 데이터셋 버전, 학교급·학년·교과가 모두 있어야 한다.
- 학교가 보유한 교육과정 원문이나 양식을 추가할 때는 배포 권한과 최신 버전을 사용자가 확인해야 한다.

## 데이터셋 갱신 규칙

1. 새 파일은 새 `datasetVersion`으로 추가하고 기존 버전을 덮어쓰지 않는다.
2. 가져오기 전에 필수 필드와 원문·해석 분리를 검증한다.
3. 기존 승인 계획은 당시의 교육과정 참조 버전을 유지한다.
4. 공개 배포 전 NCIC 고시·정정 공고와 적용 시점을 다시 확인한다.
